# 🤖 Bot de Atendimento com IA

Projeto de automação de atendimento via WhatsApp utilizando inteligência artificial.

## 🚀 Funcionalidades

- Atendimento automatizado 24/7
- Respostas com inteligência artificial
- Menu interativo por setores
- Direcionamento de clientes
- Tratamento de falhas da IA

## 🧠 Tecnologias

- Node.js
- Integração com APIs
- Automação de processos
- Inteligência Artificial

## 🎯 Objetivo

Demonstrar a criação de soluções de atendimento automatizado com foco em eficiência e experiência do usuário.

## ⚠️ Observação

Este projeto é uma versão demonstrativa para fins de portfólio, sem dados reais de empresa.
